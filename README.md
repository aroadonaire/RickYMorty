# Rick Y Morty: Guía de episodios

## Introducción
Esta app ha sido creada para la asignatura de PMDM del FP de DAM. Su propósito es peritir a los usuarios
mirar los episodios de dicha serie, saber cuales ha visto y tener las estadísticas de cuantos episodios le quedan.

## Características principales
* Un sistema de registro e inicio de sesión mediante Firebase
* Un listado de episodios que guarda en la API cuales has marcado como vistos
* Un filtrado de episodios tanto elegir todos los episodios como ver solo los que has visto ya
* Detalles de cada episodio con los personajes que aparecen en él
* Unas estadísticas con una gráfica circular que muestra el % de la seríe que ha visto la persona
* Ajustes donde he metido el cierre de sesión

## Tecnologías utilizadas
* Kotlin como lenguaje principal 
* Firebase con Auth y Firestone para la gestión de usuarios y episodios vistos
* Retrofit para la comunicación con la REST API
* Jecpack Navigation mediante NavGraph para el flujo de pantallas

## Instrucciones de uso
* Clonar el repositorio: https://github.com/aroadonaire/RickYMorty
* Configurar Firebase, es necesario añadir 'goole-services.json'
* Ejecución, abrir Android Studio, sincronizar Gradle y ejecutar
